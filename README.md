Campaign Tracker
# Objective

Campaign Tracker is a small yet powerful web app that lets you add, view, search, update, and delete marketing campaigns. It's designed to give a simple dashboard experience while showing how frontend and backend can communicate seamlessly.

# Tech Stack

Frontend: React.js – interactive and dynamic UI

Backend: Node.js + Express – handles requests and serves data

Database: JSON file (backend/campaigns.json) – simple persistent storage

Features:

Dummy login (username: admin / password: admin)

Search & filter campaigns

Dashboard summary of all campaigns

# Run Locally

---> Backend

cd backend

npm install

npm start

Backend server runs at http://localhost:5000

---> Frontend

cd frontend

npm install

npm start

wait for minute

Frontend runs at http://localhost:3000

Wait a few seconds for the frontend to connect to the backend API
